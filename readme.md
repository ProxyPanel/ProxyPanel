## Requirements
- Memory 1G +
- Disk 10G +

### Laravel 7
- PHP >= 7.2.5 && < 8.0
- BCMath PHP Extension
- Ctype PHP Extension
- Fileinfo PHP extension
- JSON PHP Extension
- Mbstring PHP Extension
- OpenSSL PHP Extension
- PDO PHP Extension
